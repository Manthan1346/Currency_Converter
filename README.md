# Currency-Converter
A simple and responsive Currency Converter built using HTML, CSS, and JavaScript.  
It fetches real-time exchange rates from an API and allows users to convert between different currencies instantly.

## Features
- Real-time currency conversion using ExchangeRate API
- Dynamic currency dropdown with country flags
- Swap currencies feature
- Automatic conversion on page load
- Highlighted exchange rate result
- Clean card-style modern UI
  
## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- ExchangeRate API
- FlagsAPI for country flags

## How to Run the Project
1. Clone this repository
2. Open the project folder
3. Open `index.html` in your browser

No installation is required.
